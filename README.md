# API Médicos e Pacientes

Uma API para gerenciar médicos e pacientes, incluindo cadastro, busca, edição e exclusão de registros.

## Tecnologias Utilizadas

- Java 21
- Spring Boot
- Maven
- Hibernate
- JPA
- MySQL

## Arquitetura

Este projeto utiliza a arquitetura MVC.
